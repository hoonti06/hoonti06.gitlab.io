## 0. 개요
'write once run anywhere'

## 1.1 데이터 타입
- 기본형(primitive type) 8개
	- 논리형 : boolean(1)
	- 문자형 : char(2)
	- 정수형 : byte(1), short(2), int(4), long(8)
	- 실수형 : float(4), double(8)
	- Call by value
- 참조형(reference type)
	- 기본형 외의 모든 타입(String, Class, Interface, Array 등)
	- 객체의 주소(4byte)
	- Call by reference
