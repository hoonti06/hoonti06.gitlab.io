## 1 번째 시도
- 점화식
	- dp[MAX_N][MAX_M]
	- dp[i][j] : j개 정수로 합 i를 만드는 경우의 수 (0도 쳐준다)
	- dp[i][j] = sum(dp[i-k][j-1]) (0 <= k <= i)
	```cpp
	for (int i = 0; i < N; i++)
		dp[i][1] = 1;
	
		for (int j = 2; j <= M; j++)
			for (int i = 0; i <= N; i++)
				for (int k = 0; k <= i; k++)
					dp[i][j] += dp[i-k][j-1];
	```
