## 
- 몫이 0이 될 때까지 -2로 나누고 양수인 나머지를 저장해둔 뒤 역순으로 출력한다.  
	
```js
input : -13
-13 / -2 =  7...1
  7 / -2 = -3...1
 -3 / -2 =  2...1
  2 / -2 = -1...0
 -1 / -2 =  1...1
  1 / -2 =  0...1
output : 110111
```
